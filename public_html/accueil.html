Bienvenue

Bienvenue sur le site officiel du projet LUD'VIN.

Le projet LUD'VIN est un ensemble d'applications pour smartphones dont le but est d'aider les personnes aveugles et malvoyantes à se localiser à l'intérieur d'un bâtiment. Il a été développé dans le cadre du projet de fin d'études de quatre étudiants du Master HANDI de l'université Paris 8.

Pour découvrir en détail le projet, rendez-vous sur la page présentation.

La page téléchargement contient les dernières versions des applications LUD'VIN pour le système Android.

Enfin sur la page liens, vous trouverez une liste de projets d'autres étudiants du Master HANDI ainsi que divers liens concernant le handicap et les nouvelles technologies.
